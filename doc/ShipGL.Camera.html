Class: Camera

ShipGL. Camera

A ShipGL.Camera object provides a means to navigate the scene.

new Camera

Creates a new ShipGL.Camera instance.
Parameters:
Name Type Description
position vec3 The camera's position
target vec3 The point the camera should be looking at
up vec3 The camera's up vector. The up you pass in will not necessarily be the camera's up vector. The passed in up is just used to help construct the camera's coordinate frame.
