Source:
  • Texture.js, line 122
Example
// Assume tex is an instance of ShipGL.Texture. You can bind the
// texture like this:
tex.bind();

// You can bind the texture and set the active texture unit in the
// same call like this:
tex.bind(0);

// The above call binds the texture and makes texture unit TEXTURE0 active.
